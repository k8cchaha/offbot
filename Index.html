使用教學

1. 如何取得 Slack 發送通知 Channel ID

  1. 在 Slack 開啟目標 Channel
  2. 點擊右上角 ⋯（更多動作），開啟頻道詳細資訊
  3. 捲到最下方，即可看到 頻道識別碼（開頭為 C）
  4. 複製該 ID 貼入設定欄位

2. 如何取得 Slack 通知對象 User ID

  1. 在 Slack 點擊目標對象的頭像或名稱，開啟個人檔案
  2. 點擊右上角的 ⋯（更多選項）
  3. 選擇 複製成員識別碼
  4. 複製的內容即為 User ID（開頭為 U）
